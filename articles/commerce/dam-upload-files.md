# <a name="upload-files-other-than-images-and-videos"></a>تحميل ملفات غير صور ومقاطع فيديو

[!include [banner](includes/banner.md)]

يصف هذا الموضوع كيفية تحميل ملفات غير الصور ومقاطع الفيديو في منشئ موقع Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>نظرة عامة

تدعم مكتبة وسائط منشئ موقع Commerce تحميل أصول ثنائية غير الصور أو مقاطع الفيديو. على سبيل المثال، قد تريد تحميل ملفات Microsoft Excel أو Microsoft Word أو Microsoft PowerPoint أو PDF.

أنواع الملفات التالية مدعومة:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>تحميل ملف

لتحميل ملف إلى منشئ موقع Commerce، اتبع الخطوات التالية.

1. في جزء التنقل الأيسر، حدد **مكتبة الوسائط**.
1. في شريط الأوامر، حدد **تحميل \> تحميل عناصر الوسائط**.
1. في "مستكشف الملفات"، حدد ملفًا أو أكثر، ثم حدد **فتح**.
1. في مربع الحوار **تحميل عنصر الوسائط**، أدخل بيانات تعريف العنوان والوصف والكلمات الأساسية كما تقتضي الحاجة.
1. لنشر الملف (الملفات) مباشرةً بعد التحميل، حدد خانه الاختيار **نشر عناصر الوسائط بعد التحميل**.
1. حدد **موافق**.
