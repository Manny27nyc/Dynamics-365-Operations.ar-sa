# <a name="store-selector-module"></a>الوحدة النمطية لمحدد المتجر

[!include [banner](includes/banner.md)]

يتناول هذا الموضوع وحدة محدّد المتجر ويصف كيفية إضافتها إلى صفحات الموقع في Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>نظرة عامة

يتم استخدام وحدة محدِّد المتجر لسيناريو العميل "الشراء عبر الإنترنت، الانتقاء في المتجر" "(BOPIS). فهو يعرض قائمه بالمتاجر التي يكون فيها المنتج متوفرًا للالتقاط، وكذلك تخزين الساعات ومخزون المنتجات لكل متجر.

تتطلب وحدة محدد المخزن الحصول على سياق منتج وموقع بحث للعثور على المتاجر. وفي حاله غياب أحد مواقع البحث، يتم تعيينه افتراضيا إلى موقع المستعرض الخاص بالعميل بشرط منح العميل الموافقة. تحتوي الوحدة على مربع إدخال يسمح للعميل بإدخال موقع (الرمز البريدي أو مدينه أو ولاية) للعثور على المتاجر القريبة.

تتكامل وحدة محدد المتجر مع واجهة برمجة تطبيقات (API) للتشفير الجغرافي لخرائط Bing لتحويل الموقع الذي ادخله العميل إلى خط عرض وخط طول. يلزم إدخال مفتاح API لخرائط Bing ويجب إضافته إلى صفحة المعلمات المشتركة للتجارة في Dynamics 365 Commerce.

يمكن إضافة وحدة محدّد المتجر إلى وحدة مربع الشراء في صفحة تفاصيل المنتج (PDP) لعرض المتاجر التي يكون فيها المنتج متوفرًا للالتقاط. ويمكن إضافته أيضًا إلى وحدة سلة. وعند إضافتها إلى وحدة السلة، تعرض وحدة محدد المتجر خيارات الانتقاء لكل صنف من بنود كل سلة. بالإضافة إلى ذلك، مع الترميز المخصص يمكن إضافة هذه الوحدة إلى صفحات أو وحدات أخرى من خلال الملحقات والتخصيصات.

لكي يعمل سيناريو BOPIS، يجب تكوين المنتجات باستخدام وضع التسليم "الانتقاء الخاص بالعملاء". وإلا، فلن تظهر الوحدة في الصفحات المعنية. للحصول على تفاصيل حول كيفية تكوين وضع التسليم، راجع [إعداد أوضاع التسليم](https://docs.microsoft.com/dynamicsax-2012/appuser-itpro/set-up-modes-of-delivery).

## <a name="store-selector-module-usage-in-e-commerce"></a>استخدام وحدة محدِّد المتجر في التجارة الإلكترونية

- يمكن استخدام وحدة محدّد المتجر في وحدة تفاصيل المنتج (PDP) للبحث عن المتاجر القريبة التي يكون فيها المنتج متوفرًا للالتقاط.
- يمكن استخدام وحدة محدّد المتجر في صفحة سلة للبحث عن المتاجر القريبة التي يكون فيها المنتج الموجود في سلة متوفرًا للالتقاط.

## <a name="store-selector-module-properties"></a>خصائص وحدة محدِّد المتجر

| اسم الخاصية             | قيمة                 | ‏‏الوصف |
|---------------------------|-----------------------|-------------|
| نصف قطر البحث | رقم | تحديد نصف قطر البحث للمتاجر، بالأميال. إذا لم يتم تحديد قيمة، يتم استخدام نصف قطر البحث الافتراضي، 50 ميلاً.|
|شروط الخدمة | URL    |  عنوان URL لشروط الخدمة المطلوب لخدمة خرائط Bing. |

## <a name="add-a-store-selector-module-to-a-page"></a>إضافة وحدة محدِّد متجر إلى صفحة

تحتاج وحدة محدِّد المتجر إلى سياق منتج، لذا يمكن استخدامه فقط داخل مربع الشراء ووحدات السلال. لا تعمل وحدات محدد المتجر خارج هذه الوحدات.

- للحصول على معلومات حول كيفية إضافة وحدة محدِّد المتجر إلى وحدة مربع شراء، راجع [وحدة مربع الشراء](add-buy-box.md). 
- للحصول على معلومات حول كيفية إضافة وحدة محدِّد المتجر إلى وحدة السلة، راجع [وحدة السلة](add-cart-module.md)
