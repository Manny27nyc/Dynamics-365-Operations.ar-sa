# <a name="manage-user-roles-in-microsoft-teams"></a>إدارة أدوار المستخدمين في Microsoft Teams

[!include [banner](includes/banner.md)]

يصف هذا الموضوع كيفية إدارة أدوار مستخدم Microsoft Dynamics 365 Commerce في Microsoft Teams.

أثناء قيامك بإنشاء فريق لكل متجر أو قناة في Teams، يتم إنشاء عضوية مجموعة تتوافق مع الفريق (على سبيل المثال،`HOUSTON_D365@<YourTenantAzureADDomain>.com`). يتم تعيين أحد أدوار المستخدمين لجميع عمال المتجر التابعين لعضوية مجموعة الفريق: **المالك** أو **عضو**. يمكن ان يقوم متجر الموظفين الذين لديهم دور المستخدم **المالك** باجراء عمليات مثل أضافه قناه خاصه وأضافه الأعضاء أو حذفهم. عادةً ما يكون لمديري المتجر دور مستخدم **المالك**.

يوضح الرسم التوضيحي التالي مثالاً لقائمة أعضاء الفريق وأدوار المستخدمين الخاصة بهم في مركز إدارة Microsoft Teams.

![أعضاء الفريق وأدوار المستخدمين في مركز إدارة Microsoft Teams.](media/d365-commerce-teams-integration-user-roles.png)

لمزيد من المعلومات، راجع [تعيين مالكي الفريق وأعضائه في Microsoft Teams](/microsoftteams/assign-roles-permissions).
