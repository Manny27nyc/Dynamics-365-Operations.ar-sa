# <a name="inventory-aging-report-discrepancies"></a>تباينات تقرير تقادم المخزون

## <a name="symptoms"></a>العلامات

يعرض **تقرير تقادم المخزون** قيما مختلفه عند عرضها علي ابعاد تخزين مختلفه (مثل الموقع أو المستودع).

## <a name="resolution"></a>الحل

لمزيد من المعلومات حول منطق التقرير ، راجع [أمثله تقرير فتره تاخر المخزون ومنطقه](/dynamics365/supply-chain/cost-management/inventory-aging-report.md).
