# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>لم يتم تحديث اسم المورد بعد تغيير اسم العامل في دفتر العناوين

## <a name="symptoms"></a>العلامات

إذا قمت بتغيير اسم العامل في دفتر العناوين العمومي، لا يتم تحديث اسم المورد المطابق في السجل الرئيسي لمجموعة الموارد.

## <a name="resolution"></a>الحل

لا يتم تحديث اسم المورد المطابق بشكل تلقائي في الوقت الحالي. لإصلاح المشكلة ، يجب تحديث اسم المورد يدويا.
