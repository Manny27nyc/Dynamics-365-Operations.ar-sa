# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>يمكن فقط استخدام دور المسؤول للوصول إلى محاكي تطبيق الأجهزة المحمولة

## <a name="symptoms"></a>العلامات

عند إعداد المستودعات في Supply Chain Management، لا يمكنك استخدام أي دور باستثناء دور المسؤول للوصول إلى محاكي "تطبيق الأجهزة المحمولة".

## <a name="resolution"></a>الحل

يتم تعيين محاكي تطبيق الجهاز المحمول للعمل فقط مع حساب المسؤول. بالنسبة لكافة أغراض الاختبار والمعالجة المباشرة ، نوصي باستخدام تطبيق إدارة المستودع للأجهزة المحمولة نفسه.
