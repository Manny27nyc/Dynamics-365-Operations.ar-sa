# <a name="packing-station-doesnt-show-product-notes"></a>لا تعرض محطة التعبئة ملاحظات المنتج

رقم KB: 4614615

## <a name="symptoms"></a>العلامات

لا يتم عرض ملاحظات التعبئة في نموذج التعبئة عند إضافة إرشادات التعبئة كمرفق إلى أصل منتج أو متغير منتج.

## <a name="resolution"></a>الدقة

يعمل النظام كما هو مصمم.

يتطلب المنطق الحالي الخاص بعرض إشعارات التعبئة في نموذج التعبئة أن تكون الإشعارات مقترنة بالشحنات.
