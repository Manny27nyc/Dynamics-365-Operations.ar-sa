# <a name="create-a-new-kanban-rule-by-duplicating-an-existing-kanban-rule"></a>إنشاء قاعدة كانبان جديدة عن طريق تكرار قاعدة كانبان الموجودة

[!include [task guide banner](../../includes/task-guide-banner.md)]

ويركز هذا الإجراء على إنشاء نسخة مكررة من قاعدة كانبان موجودة. وهذا مفيد إذا كنت تريد إنشاء قواعد كانبان جديدة تستند إلى قواعد كانبان الموجودة. شركة بيانات العرض التوضيحي التي تم استخدامها لإنشاء هذا الإجراء هي USMF. هذا الإجراء مخصص لمهندس العمليات أو مدير تدفق القيم عند تحضيرهم لإنتاج تدفق عمل مغيَّر أو قاعدة تزويد جديدة.


## <a name="select-a-kanban-rule"></a>حدد "قاعدة كانبان".
1. انتقل إلى قواعد كانبان.
2. في القائمة، قم بالبحث عن السجل المطلوب وحدده.
    * حدد "قاعدة كانبان 000017 للمنتج M0006".  

## <a name="duplicate-a-kanban-rule"></a>تكرار قاعدة كانبان
1. انقر فوق "تكرار قاعدة كانبان".
    * عند تكرار قاعدة كانبان، من الممكن تغيير تحديد النوع والتواريخ والأنشطة والمنتج. قم بتغيير المنتج المخصص لهذا الإجراء في الخطوة التالية.  
2. في الحقل "المنتج"، أدخل قيمة أو حددها.
    * حدد "M0007".  
3. انقر فوق "موافق".
    * لاحظ أنه يتم إنشاء نسخة مكررة قاعدة كانبان 000017.
