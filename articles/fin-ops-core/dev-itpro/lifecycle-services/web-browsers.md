# <a name="supported-web-browsers-for-lifecycle-services"></a>‏‫مستعرضات الويب المدعومة لـ Lifecycle Services 

[!include[banner](../includes/banner.md)]

يمكن للمستخدمين الوصول إلى خدمات Microsoft Dynamics Lifecyle Services (LCS) باستخدام أحدث إصدارات المستعرضات الشائعة: 

- Microsoft Edge(موصى به: [Chromium-based Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

للعثور على أحدث إصدار لكل مستعرض ويب، انتقل إلى موقع ويب الشركة المصنعة للبرنامج. للحصول على أفضل أداء وتجربة مثالية، نوصي باستخدام أحدث إصدار من المستعرض الحديث، وخاصة Microsoft Edge.
