# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-october-15-2018"></a>ما الجديد أو المتغير في Dynamics 365 for Talent Core HR‏ (15 أكتوبر 2018)

[!include [banner](includes/banner.md)]

**الإصدار 8.1.1056**

يصف هذا الموضوع الميزات الجديدة أو المتغيرة في Core HR.


## <a name="changes"></a>التغييرات
بالإضافة إلى الإصلاحات المتفرقة، تم إجراء التحديثات التالية في هذا الإصدار:
- يتم الآن تعيين آخر يوم عمل‬ عند التوظيف أو تعيين تاريخ انتهاء توظيف.
- تمت إزالة مراجع التوافق الأمريكية في الشركات غير الأمريكية (ACA وADA وI9).
- التواريخ غير الصالحة (1/1/1900) أصبحت الآن مخفية في صفحات التحليلات.

## <a name="known-issue"></a>مشكلات معروفة​

**مشكلة:** عند إضافة مرفق جديد لعامل، تتحول الأزرار **جديد** و **تحرير** إلى اللون الرمادي. **حل بديل:** قبل فتح صفحة المرفق، تأكد أن FactBoxes في صفحة **العامل** مغلق. إذا كانت مربعات الحقائق مغلقة عند تحميل صفحة **العامل**، سيتم تمكين زر المرفقات (سوف يتم إصلاح هذه المشكلة في التحديث التالي للنظام الأساسي.)
