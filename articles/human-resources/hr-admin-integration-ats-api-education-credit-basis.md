[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

يصف هذا الموضوع مجموعة خيارات أساس الساعات المعتمدة للتعليم لـ Dynamics 365 Human Resources.

الاسم الفعلي: mshr_hcmeducationcreditbasis

توفر قائمة التعداد هذه مجموعة خيارات قيم أساس الساعات المعتمدة بلتعليم للسجل التعليمي الخاص بالمرشح.

| قيمة | تسمية | الوصف |

| 200000000 | فارغ | لم يتم تحديد أي قيمة. |
| 200000001 | نصف السنة | نصف السنة. |
| 200000002 | ربع السنة | ربع السنة. |
| 200000003 | فصل ربع سنوي | ثلث السنة. |
| 200000004 | الفصل الدراسي | الفصل الدراسي. |
| 200000005 | أخرى | أخرى. |
