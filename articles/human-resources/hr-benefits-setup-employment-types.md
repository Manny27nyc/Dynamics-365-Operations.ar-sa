# <a name="set-up-employment-types"></a>إعداد أنواع التوظيف

[!include [banner](includes/preview-feature.md)]

قم بإعداد أنواع التوظيف في Dynamics 365 Human Resources لمطابقة العاملين.

1. في مساحة العمل **إدارة الميزات**، ضمن **إعداد**، حدد **أنواع التوظيف**.

2. حدد **جديد**.

3. حدد قيمًا للحقول التالية:

   | الحقل | ‏‏الوصف |
   | --- | --- |
   | نوع التوظيف | تعريف فريد لنوع التوظيف. |
   | ‏‏الوصف | وصف نوع التوظيف. |

4. حدد **حفظ**.
