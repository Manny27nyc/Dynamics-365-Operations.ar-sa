# <a name="set-up-tier-codes"></a>إعداد أكواد الطبقات

يستخدم Dynamics 365 Human Resources أكواد الطبقات بأسعار المزايا التي يدفعها الموظفون وأصحاب العمل. يمكنك تحديد عدد غير محدود من أكواد الطبقات وإعداد حسابات المعدلات لتلبية احتياجات شركتك.

1. في مساحة العمل **إدارة الميزات**، ضمن **إعداد**، حدد **أكواد الطبقات**.

2. حدد **جديد**.

3. حدد قيمًا للحقول التالية:

   | الحقل | ‏‏الوصف |
   | --- | --- |
   | **رمز الطبقة** | كود أو رقم ميزة فريد لتحديد كود الطبقة. |
   | **‏‏الوصف** | وصف موجز لكود طبقة الميزة. |

4. حدد **حفظ**.
