| قيمة | تسمية | الوصف |
| --- | --- | --- |
| 200000000 | ‏‫لا شيء‬ | بلا. |
| 200000001 | سنوي | سنوي. |
| 200000002 | الذكرى السنوية | الذكرى السنوية. |
| 200000003 | غير دوري | غير دوري. |
| 200000004 | ½ عام | نصف سنوي. |
| 200000005 | ربع سنوي | ربع سنوي. |
| 200000006 | شهري | شهري. |
