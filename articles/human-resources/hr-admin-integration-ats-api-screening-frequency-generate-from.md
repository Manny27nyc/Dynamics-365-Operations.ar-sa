# <a name="screening-frequency-generate-from"></a>إنشاء تكرار المراقبة من

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

يصف هذا الموضوع إنشاء تكرار المراقبة من مجموعة الخيارات لـ Dynamics 365 Human Resources.

الاسم الفعلي: mshr_hcmfrequencygeneratefrom

يوفر هذا التعداد مجموعة خيارات لقيم تحديد تاريخ بدء عملية الحساب لعملية المراقبة المطلوبة التالية.

| قيمة | تسمية | الوصف |
| --- | --- | --- |
| 200000000 غير محدد | لم يتم تحديد أي قيمة. |
| 200000001 تاريخ الإكمال | يتم إجراء عملية الحساب من تاريخ آخر عملية مراقبة تم إكمالها. |
| 200000002 التاريخ المطلوب | يتم إجراء عملية الحساب من تاريخ آخر عملية مراقبة مطلوبة. |
