# <a name="award-type"></a>نوع المكافأة

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

يصف هذا الموضوع مجموعة خيارات نوع المكافأة لـ Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>مجموعة الخيارات: نوع المكافأة

الاسم الفعلي: mshr_HrmCompVarAwardEmplType

يوفر هذا التعداد مجموعة الخيارات لنوع مكافأة التعويض المتغير في الكيانات الظاهرية للموارد البشرية.

| قيمة | تسمية | الوصف |
| --- | --- | --- |
| 200000000 | المكافأة العامة | المكافأة العامة. |
| 200000001 | المبلغ الإجمالي للدفع الثابت | المبلغ الإجمالي للدفع الثابت. |
