# <a name="contact-type"></a>نوع جهة الاتصال

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

يصف هذا الموضوع مجموعة خيارات نوع جهة الاتصال لـ Dynamics 365 Human Resources.

الاسم الفعلي: mshr_logisticselectronicaddressmethodtype

توفر قائمة التعداد هذه مجموعة الخيارات الخاصة بقيم أنواع جهات الاتصال بالمرشحين. 

| قيمة | تسمية | الوصف |
| --- | --- | --- |
| 200000000 | لا شيء | لم يتم تحديد أي نوع. |
| 200000001 | الهاتف | رقم الهاتف. |
| 200000002 | عنوان البريد الإلكتروني | عنوان البريد الإلكتروني. |
| 200000003 | URL | عنوان URL لموقع الويب. |
| 200000004 | تلكس | رقم التلكس. |
| 200000005 | فاكس | رقم الفاكس. |
| 200000006 | Facebook | حساب Facebook. يتم تحديده بواسطة معرف المستخدم. |
| 200000007 | Twitter | حساب Twitter. يتم تحديده بواسطة @username. |
| 200000008 | LinkedIn | حساب LinkedIn. يتم تحديده بواسطة اسم المستخدم. |
