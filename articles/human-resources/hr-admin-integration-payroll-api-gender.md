توفر قائمة التعداد هذه مجموعة الخيارات الخاصة بأجناس الموظفين. ويتوفر ذلك في مجموعة الخيارات mshr_hcmpersongender.

| قيمة | تسمية | الوصف |
| --- | --- | --- |

| 200000001 | ذكر | ذكر. |
| 200000002 | أنثى | أنثى. |
| 200000003 | غير محدد | التحديد لنوع غير خاص. |
