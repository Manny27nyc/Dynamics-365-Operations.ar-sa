# <a name="screening-frequency"></a>تكرار المراقبة

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

يصف هذا الموضوع مجموعة خيارات تكرار المراقبة لـ Dynamics 365 Human Resources.

الاسم الفعلي: mshr_hcmfrequencyunit

توفر قائمة التعداد هذه مجموعة الخيارات الخاصة بقيم تكرار المراقبة. 

| قيمة | تسمية | الوصف |
| --- | --- | --- |
| 200000000 مرة واحدة فقط | تكون عملية المراقبة مطلوبة مرة واحدة فقط. |
| 200000001 يوميًا | يتم حساب تكرار المراقبة بالأيام. |
| 200000002 أسبوعيًا | يتم حساب تكرار المراقبة بالأسابيع. |
| 200000003 شهريًا | يتم حساب تكرار المراقبة بالشهور. |
| 200000004 سنويًا | يتم حساب تكرار المراقبة بالسنوات. |
