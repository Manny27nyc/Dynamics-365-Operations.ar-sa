# <a name="skill-level-type"></a>نوع مستوى المهارة


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

توضح هذه المقالة مجموعة خيارات نوع مستوى المهارة في Dynamics 365 Human Resources.

الاسم الفعلي: mshr_hrmskillleveltype

توفر قائمة التعداد هذه مجموعة خيارات القيم التي تقوم بتصنيف المستوى الذي يتم تعيينه لمهارة المرشح.

| قيمة | تسمية | الوصف |
| --- | --- | --- |
| 200000000 | القيم الفعلية | مستوى المهارة الفعلي. |
| 200000001 | الهدف | القيمة المستهدفة لمستوى المهارة. |
