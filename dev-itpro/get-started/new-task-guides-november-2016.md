# <a name="new-or-updated-task-guides-november-2016"></a>أدلة مهمة جديدة أو محدثة (تشرين الثاني/نوفمبر 2016)

يسرد هذا الموضوع الجديد أو المحدث المهمة الأدلة التي تم إصدارها إلى Microsoft (تشرين الثاني/نوفمبر عام 2016-جميع اللغات) APQC الموحدة مكتبة Microsoft Dynamics 365 للعمليات. لم يتم إصدار مكتبة بدء استخدام جديدة.

(تشرين الثاني/نوفمبر 2016-جميع اللغات) مكتبة الموحدة APQC يحتوي على إرشادات المهمة التي يعتزم استخدامها للحصول على تعليمات Microsoft Dynamics 365 لعمليات تحديث النظام الأساسي 3، تحديث النظام الأساسي ل Microsoft Dynamics AX 2، Dynamics AX، Microsoft Dynamics 365 لعمليات إصدار تحديث 2016 مايو و Microsoft Dynamics AX 2016 شباط/فبراير. لمزيد من المعلومات حول هذه الإصدارات، راجع [ما هو جديد أو تغيير](whats-new-changed.md). تتوفر الإصدارات المترجمة من الأدلة المهمة في هذه المكتبة. لمزيد من المعلومات حول كيفية توصيل نظام التعليمات إلى مكتبة دليل مهام، راجع [الشروع في تعليمات](help-overview.md). **ملاحظة:** علامة نجمة (**\***) بجوار عنوان مصمم نماذج عملية الأعمال (BPM) البند إلى أنه تم إنشاء دليل المهمة لهذه العملية التجارية لتحديث Dynamics 365 لعمليات 2016 تشرين الثاني/نوفمبر، وغير معتمدة في الإصدارات السابقة من Microsoft Dynamics AX.
