# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>حساب إهلاك الأصول الثابتة عبر الكيانات القانونية

[!include[task guide banner](../../includes/task-guide-banner.md)]

استخدم هذا الإجراء لتغيير مجموعة الأصول الثابتة التي تم تعيين أصل ثابت لها. يجب تعيين الأصول الثابتة إلى مجموعة الأصول الثابتة الصحيحة. يتم استخدام مجموعة الأصول الثابتة عند إنشاء الاستعلامات والتقارير، وإعداد الأصول الثابتة الجديدة، ودمج دفاتر الأستاذ وترحيل حركات الأصول الثابتة لحسابات دفتر الأستاذ المناسبة.

يستخدم هذا التسجيل شركة بيانات العرض التوضيحي USMF.

1. انتقل إلى الأصول الثابتة > الأصول الثابتة > الأصول الثابتة.
2. حدد الأصل الثابت الذي تريد تغيير مجموعة الأصول الثابتة له.
3. انقر فوق "تغيير مجموعة الأصول الثابتة".
4. في الحقل "مجموعة جديدة"، أدخل قيمة أو حددها.
5. عيّن خيار "رقم الأصل الثابت الجديد" إلى "نعم" لتعيين رقم أصل ثابت إلى الأصل الثابت المحدد.
    * يصبح حقل "رقم الأصل الثابت‬" متوفرًا عند تعيين الخيار "رقم الأصل الثابت الجديد‬" إلى "نعم".   وفي حالة إعداد ترقيم تلقائي للأصول الثابتة، يعرض هذا الحقل رقم الأصل الثابت المتوفر التالي. ويمكنك تغيير الرقم.   في حالة إعداد الترقيم اليدوي للأصول الثابتة، يكون هذا الحقل فارغًا ويجب إدخال رقم الأصل الثابت الجديد.  
6. انقر فوق "موافق".
7. انقر فوق نعم.
