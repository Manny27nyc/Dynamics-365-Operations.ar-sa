# <a name="publish-journal-lines-and-documents-from-excel"></a>نشر المستندات من Excel وبنود دفتر اليومية

يشرح هذا المقال كيفية إدخال ونشر بنود دفاتر اليومية العامة من Microsoft Excel. يتضمن معلومات حول القوالب المتعددة التي يمكنك استخدامها، اعتماداً على نوع الحركات التي تقوم بإدخال.

يمكن للمستخدمين إدخال ونشر بنود دفاتر اليومية المالية من Microsoft Excel. بعد أن يقوم مستخدم بإنشاء دفتر يومية **في Excel بفتح خطوط** زر عرض القوالب المتوفرة. قوالب مصممة لدعم وحدات السيناريو المحددة، يتم دعم لكن ليس كل تركيبة من نوع الحساب في دفتر اليومية. يعرض الجدول التالي القوالب المتوفرة وأنواع الحسابات التي تدعمها.
|                          |                                                                                                                         |                                                                                         |
|--------------------------|-------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| **Template**             | **أنواع الحساب المعتمدة**                                                                                             | **كيفية الوصول إلى قالب**                                                          |
| بنود دفتر يومية دفتر الأستاذ     | الحساب: دفتر الأستاذ والعميل أو المورد، الحساب "المقابل البنك": يتم اعتماد دفتر الأستاذ، العميل، المورد، البنك بين الشركات الشقيقة.       | دفتر اليومية العام                                                                         |

| دفتر يومية الفواتير          | الحسابات: حساب المورد الإزاحة: يتم اعتماد دفتر الأستاذ بين الشركات الشقيقة.                                                      | دفتر يومية فاتورة الحسابات الدائنة                                                                      |
| فاتورة المورّد           |                                                                                                                         | فاتورة المورّد                                                                          |
| دفتر يومية فواتير العميل | الحساب: حساب العميل مقابل: يتم اعتماد دفتر الأستاذ بين الشركات الشقيقة.                                                     | دفتر اليومية العام                                                                         |

| دفتر يومية المدفوعات للمورد   | الحساب: حساب المورد مقابل: يتم اعتماد دفتر الأستاذ، البنك بين الشركات الشقيقة.                                                 | دفتر يومية المدفوعات للمورد                                                                  |

| دفتر يومية مصروفات المشروع  | الحساب: المشروع، دفتر الأستاذ والعميل، الحساب "المقابل للمورد": يتم اعتماد المشروع، دفتر الأستاذ، العميل، المورد بين الشركات الشقيقة. | المصروفات اليومية العام (في إطار المحاسبة وإدارة المشروع)                       |

عند نشر البنود، يتم التحقق من صحة للتأكد من امتثالها للقواعد التي تم إعدادها في دفاتر اليومية المالية. بعد نشر البنود، يمكن للمستخدمين تحرير أو ترحيل الإيصالات من Microsoft Dynamics 365 للعمليات. 

لإضافة أبعاد مالية إلى قالب، تغييرات إضافية مطلوبة. لمزيد من المعلومات، انظر [إضافة أبعاد إلى قالب Microsoft Excel](\dev-itpro\financial-dimensions\add-dimensions-excel-templates). بعد إضافة أبعاد إلى الكيان، تتوفر في مصمم Excel ويمكن إضافتها إلى القالب.
